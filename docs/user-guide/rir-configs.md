# RIR Configs

An **RIR Config** binds the plugin to a specific RIR account.

## Creating a config

1. Navigate to **RIR Manager → Configs** and click **Add**.
2. Select the RIR backend (e.g. **ARIN**).
3. Provide the organization handle for your account.
4. Save.

Once created, the config exposes actions for syncing, write operations, and ticket tracking. Each user must additionally provide their own API key (see [User API Keys](user-api-keys.md)).
